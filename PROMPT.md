## پرامپت جامع و تفصیلی برای ارتقاء پروژه IslamNET

با توجه به توسعه‌های اخیر در پروژه IslamNET و یکپارچه‌سازی جادوگر نصب Merlin WP و بهبود مدیریت افزونه‌ها، اکنون زمان آن رسیده است که پروژه را به سطح بعدی ارتقاء دهیم. این پرامپت شامل دستورالعمل‌های دقیق برای رفع باگ‌ها، بهبود ظاهر و تجربه کاربری (UI/UX) به سبک مینیمال آیفون، و افزودن امکانات جدید و پیشرفته به شبکه اجتماعی اسلامی شماست. هدف نهایی ایجاد یک پلتفرم قدرتمند، زیبا، کاربردی و مطابق با نیازهای جامعه مسلمانان است.

### 1. رفع باگ‌ها و بهبود کدنویسی (Bug Fixing & Code Refinement)

**هدف:** اطمینان از پایداری، امنیت، عملکرد بهینه و رعایت استانداردهای کدنویسی در تمامی بخش‌های پوسته.

*   **بررسی و رفع خطاهای PHP و WordPress:**
    *   **محیط توسعه:** پوسته IslamNET را در یک محیط توسعه محلی (مانند XAMPP, MAMP, Local by Flywheel) با فعال بودن `WP_DEBUG` و `WP_DEBUG_LOG` در فایل `wp-config.php` نصب و راه‌اندازی کنید. تمامی اخطارها، خطاها و نوتیفیکیشن‌های PHP را در حین کار با پوسته (چه در بخش فرانت‌اند و چه در بخش ادمین) به دقت بررسی و برطرف نمایید.
    *   **فایل‌های کلیدی:** به طور خاص، فایل‌های `functions.php`, `install.php`, `inc/setup/class-islamnet-setup-wizard.php` و تمامی فایل‌های موجود در دایرکتوری `inc/Merlin` را برای هرگونه خطای منطقی، استفاده نادرست از توابع وردپرس یا PHP، و مسائل مربوط به عملکرد بررسی کنید.
    *   **تداخلات:** اطمینان حاصل کنید که تمامی توابع و کلاس‌ها به درستی بارگذاری می‌شوند و تداخلی با یکدیگر، با افزونه‌های دیگر (به ویژه BuddyPress, Elementor, LearnPress) یا با هسته وردپرس ندارند.
    *   **استانداردهای کدنویسی:** تمامی کدهای PHP باید از [WordPress Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/php/) پیروی کنند. از ابزارهایی مانند PHP_CodeSniffer با استاندارد WordPress برای بررسی خودکار استفاده کنید.
*   **بهینه‌سازی عملکرد (Performance Optimization):**
    *   **CSS و JavaScript:** کدهای CSS و JavaScript را برای کاهش حجم و بهبود سرعت بارگذاری بهینه‌سازی کنید (Minification و Concatenation). از Lazy Loading برای تصاویر و ویدئوها استفاده کنید. اطمینان حاصل کنید که اسکریپت‌ها و استایل‌ها فقط در صفحاتی که نیاز دارند بارگذاری شوند (Conditional Loading).
    *   **تصاویر:** تصاویر را فشرده‌سازی کرده و از فرمت‌های مدرن و بهینه (مانند WebP) استفاده کنید. قابلیت Responsive Images وردپرس را به درستی پیاده‌سازی کنید.
    *   **کوئری‌های دیتابیس:** کوئری‌های دیتابیس را در صورت لزوم بهینه‌سازی کنید تا بار روی سرور کاهش یابد. از Transient API وردپرس برای کش کردن نتایج کوئری‌های سنگین استفاده کنید.
    *   **کشینگ:** سازگاری کامل با افزونه‌های کشینگ محبوب (مانند WP Super Cache, WP Rocket) را تضمین کنید.
*   **امنیت (Security):**
    *   **اعتبارسنجی و پاک‌سازی:** تمامی ورودی‌های کاربر (مانند فرم‌ها، کامنت‌ها، فیلدهای پروفایل) را به دقت اعتبارسنجی و پاک‌سازی (Sanitize and Validate) کنید تا از حملات XSS، SQL Injection و سایر آسیب‌پذیری‌ها جلوگیری شود.
    *   **Nonce:** از Nonce برای تمامی فرم‌ها و اکشن‌های حساس (مانند ارسال فرم، حذف محتوا) استفاده کنید.
    *   **مجوزهای دسترسی:** اطمینان حاصل کنید که فایل‌ها و دایرکتوری‌ها دارای مجوزهای دسترسی صحیح هستند و اطلاعات حساس در دسترس عموم قرار نمی‌گیرند.
    *   **امنیت وردپرس:** از بهترین شیوه‌های امنیتی وردپرس پیروی کنید.

### 2. طراحی UI/UX با الهام از آیفون (مینیمال، سیاه و سفید) (iPhone-Inspired Minimal UI/UX)

**هدف:** ایجاد یک تجربه کاربری مدرن، تمیز، بصری جذاب و کاربردی با الهام از طراحی مینیمال و شیک آیفون، با پالت رنگی محدود به سیاه، سفید و طیف‌های خاکستری. این طراحی باید کاربردی، بصری و باکیفیت باشد.

*   **پالت رنگی (Color Palette):**
    *   **رنگ‌های اصلی:** استفاده غالب از رنگ‌های **سفید (`#FFFFFF`)** و **مشکی (`#000000`)** به عنوان رنگ‌های اصلی برای پس‌زمینه‌ها، متن‌ها و عناصر کلیدی.
    *   **طیف‌های خاکستری:** استفاده هوشمندانه از طیف‌های **خاکستری** برای ایجاد عمق، تفکیک بصری و برجسته‌سازی بدون افزودن رنگ‌های اضافی. مثال‌ها:
        *   **پس‌زمینه‌های روشن:** `#F2F2F7` (iOS Light Mode Background)
        *   **پس‌زمینه‌های تیره:** `#1C1C1E` (iOS Dark Mode Background)
        *   **جداکننده‌ها و حاشیه‌ها:** `#E5E5EA` (Light), `#3A3A3C` (Dark)
        *   **متن‌های ثانویه/کپشن‌ها:** `#8E8E93` (Light), `#AEAEB2` (Dark)
    *   **رنگ تاکیدی (Accent Color):** برای عناصر تعاملی (مانند دکمه‌های اصلی، لینک‌های فعال، آیکون‌های انتخاب شده، نوتیفیکیشن‌ها) از یک رنگ تاکیدی بسیار ملایم و کم‌رنگ (مانند یک آبی یا سبز بسیار مات و نزدیک به خاکستری, مثلاً `#007AFF` با شفافیت پایین یا `#34C759` با شفافیت پایین) استفاده شود. این رنگ باید به گونه‌ای باشد که با پالت سیاه و سفید هماهنگی داشته باشد و حس مینیمال را حفظ کند. جایگزین: استفاده از تغییرات ظریف در ضخامت فونت، سایه ملایم یا هایلایت خاکستری برای حالت‌های Hover و Active.
*   **تایپوگرافی (Typography):**
    *   **فونت:** استفاده از فونت‌های سیستمی iOS (مانند San Francisco Pro Display و San Francisco Pro Text) یا فونت‌های مشابه بدون سریف (مانند Inter, Roboto, SF Pro Display) که حس مدرن و خوانایی بالا را ارائه می‌دهند. برای زبان فارسی، از فونت‌های مدرن و خوانا مانند ایران سنس (Iran Sans) یا یکان بخ (Yekan Bakh) استفاده شود که با طراحی مینیمال سازگار باشند.
    *   **سلسله مراتب:** تمرکز بر خوانایی بالا با استفاده از اندازه‌ها و وزن‌های مختلف فونت برای ایجاد سلسله مراتب بصری واضح (مثلاً Bold برای عناوین، Regular برای متن اصلی، Light برای توضیحات ثانویه).
*   **فضای سفید (Whitespace):**
    *   استفاده سخاوتمندانه و متعادل از فضای سفید (Padding و Margin) برای کاهش شلوغی، بهبود خوانایی و تمرکز کاربر بر محتوا. ایجاد یک سیستم گرید (Grid System) ثابت برای حفظ یکنواختی در سراسر پوسته.
*   **عناصر رابط کاربری (UI Elements):**
    *   **دکمه‌ها:** طراحی دکمه‌های ساده، با گوشه‌های گرد ملایم (Border-radius: 8px-12px)، بدون گرادیانت‌های پیچیده یا سایه‌های سنگین. حالت‌های Hover و Active باید ظریف و مینیمال باشند (مثلاً تغییر رنگ پس‌زمینه به خاکستری روشن‌تر یا تیره‌تر، یا کاهش شفافیت).
    *   **فرم‌ها و فیلدهای ورودی:** فیلدهای ورودی تمیز، با حاشیه‌های نازک (1px solid #E5E5EA) و تمرکز بر سادگی. استفاده از Placeholder Text واضح و فونت مناسب. حالت Focus باید با یک حاشیه ظریف یا تغییر رنگ پس‌زمینه مشخص شود.
    *   **آیکون‌ها:** استفاده از آیکون‌های خطی (Outline Icons) یا آیکون‌های پر شده (Filled Icons) با طراحی مینیمال و یکنواخت (مانند SF Symbols اپل). اطمینان از سازگاری آیکون‌ها در حالت‌های روشن و تاریک.
    *   **ناوبری (Navigation):** منوهای ناوبری باید ساده، واضح و با قابلیت دسترسی آسان باشند. استفاده از منوهای همبرگری (Hamburger Menu) برای موبایل و ناوبری ثابت در بالا یا پایین صفحه (Bottom Navigation Bar) برای دسترسی سریع به بخش‌های اصلی در موبایل. در دسکتاپ، ناوبری باید مینیمال و سازمان‌یافته باشد.
    *   **کارت‌ها و لیست‌ها:** طراحی عناصر محتوایی (مانند پست‌ها، پروفایل‌ها، دوره‌ها) به صورت کارت‌های تمیز با سایه‌های ملایم یا لیست‌های ساده با جداکننده‌های ظریف.
    *   **مودال‌ها و پاپ‌آپ‌ها:** طراحی مودال‌ها و پاپ‌آپ‌ها با پس‌زمینه مات و محتوای مرکزی، با انیمیشن‌های ورود و خروج نرم.
*   **ریسپانسیو بودن (Responsiveness):**
    *   طراحی کاملاً ریسپانسیو برای تمامی دستگاه‌ها (موبایل، تبلت، دسکتاپ)، با تمرکز ویژه بر تجربه کاربری موبایل (Mobile-First Design). تمامی عناصر باید در اندازه‌های مختلف صفحه نمایش به درستی نمایش داده شوند و تعامل‌پذیری خود را حفظ کنند.
*   **انیمیشن‌ها و ترنزیشن‌ها (Animations & Transitions):**
    *   استفاده از انیمیشن‌ها و ترنزیشن‌های ظریف و روان (مانند Fade, Slide, Scale) برای بهبود تجربه کاربری و حس پویایی، مشابه انیمیشن‌های iOS. سرعت انیمیشن‌ها باید بهینه باشد تا حس سرعت و پاسخگویی را حفظ کند.
*   **حالت تاریک (Dark Mode):**
    *   پیاده‌سازی کامل و بومی حالت تاریک (Dark Mode) که تمامی عناصر UI/UX را به پالت رنگی تیره (مشکی و خاکستری تیره) تبدیل کند. این حالت باید به صورت خودکار بر اساس تنظیمات سیستم عامل کاربر فعال شود و همچنین یک گزینه دستی برای تغییر حالت در دسترس باشد.

### 3. افزودن امکانات اولیه و ثانویه به شبکه و پیشنهادات تکمیلی (Core & Advanced Features + Agent's Suggestions)

**هدف:** گسترش قابلیت‌های IslamNET برای تبدیل شدن به یک پلتفرم جامع، جذاب و نوآورانه برای جامعه مسلمانان، با تمرکز بر نیازهای مذهبی، اجتماعی و آموزشی.

#### 3.1. امکانات اولیه (Core Features) - بهبود و توسعه BuddyPress و LearnPress

*   **سیستم پیام‌رسانی پیشرفته (Enhanced Messaging System):**
    *   **پیام‌های خصوصی:** امکان ارسال پیام‌های خصوصی یک به یک و گروهی با قابلیت ارسال فایل (اسناد، تصاویر، ویدئوهای کوتاه) و ایموجی.
    *   **نوتیفیکیشن‌های لحظه‌ای:** پیاده‌سازی نوتیفیکیشن‌های لحظه‌ای (Real-time Notifications) برای پیام‌های جدید، درخواست‌های دوستی و فعالیت‌های گروهی.
    *   **مدیریت پیام:** قابلیت آرشیو، حذف، مسدود کردن کاربران و گزارش پیام‌های نامناسب.
    *   **ادغام با BuddyPress:** استفاده از قابلیت‌های پیام‌رسانی BuddyPress و بهبود UI/UX آن برای مطابقت با طراحی آیفون.
*   **سیستم رویدادها و تقویم اسلامی (Islamic Events & Calendar System):**
    *   **ایجاد و مدیریت رویدادها:** کاربران و مدیران بتوانند رویدادهای اسلامی (مانند نماز جمعه، جلسات قرآن، کنفرانس‌ها، مراسم مذهبی) را ایجاد، مدیریت و نمایش دهند.
    *   **تقویم شمسی و قمری:** نمایش تقویم شمسی و قمری با قابلیت تبدیل تاریخ‌ها و نمایش مناسبت‌های مهم اسلامی (مانند اعیاد، وفیات، ایام خاص).
    *   **اوقات شرعی و قبله‌نما:** ادغام با APIهای معتبر برای نمایش دقیق اوقات شرعی بر اساس موقعیت جغرافیایی کاربر و قابلیت قبله‌نما.
    *   **یادآوری:** امکان تنظیم یادآوری برای رویدادها و اوقات شرعی.
*   **پروفایل‌های کاربری غنی (Rich User Profiles):**
    *   **اطلاعات مذهبی:** قابلیت افزودن فیلدهای سفارشی برای اطلاعات مذهبی (مانند مذهب، گرایش فکری، میزان آشنایی با قرآن و حدیث، اساتید، کتب مورد علاقه).
    *   **معرفی و علاقه‌مندی‌ها:** بخش معرفی خود، علاقه‌مندی‌ها، مهارت‌ها و اهداف معنوی.
    *   **فعالیت‌ها و دنبال‌کنندگان:** نمایش فعالیت‌های عمومی کاربر، تعداد دنبال‌کنندگان و دنبال‌شونده‌ها.
    *   **حریم خصوصی:** تنظیمات حریم خصوصی پیشرفته برای کنترل نمایش اطلاعات پروفایل.
*   **سیستم گروه‌ها و انجمن‌ها (BuddyPress Enhancement):**
    *   **مدیریت محتوا و اعضا:** بهبود قابلیت‌های گروه‌های BuddyPress با امکانات پیشرفته‌تر برای مدیریت محتوا (پست‌ها، فایل‌ها)، اعضا و سطوح دسترسی.
    *   **انجمن‌های تخصصی:** ایجاد انجمن‌های تخصصی برای مباحث فقهی، تفسیری، اخلاقی، تاریخی و سایر علوم اسلامی با قابلیت دسته‌بندی، جستجو و فیلتر.
    *   **رابط کاربری:** بازطراحی رابط کاربری گروه‌ها و انجمن‌ها برای مطابقت با سبک مینیمال آیفون.

#### 3.2. امکانات ثانویه (Advanced Features) - ماژول‌های توسعه‌یافته

*   **بازارچه اسلامی (Islamic Marketplace - WooCommerce Integration):**
    *   **فروش محصولات حلال:** امکان فروش محصولات حلال، کتاب‌های اسلامی، لباس‌های محجبه، صنایع دستی اسلامی و خدمات مرتبط (مانند مشاوره مذهبی، کلاس‌های آنلاین).
    *   **سیستم پرداخت:** یکپارچه‌سازی با سیستم‌های پرداخت امن و محبوب (هم بین‌المللی و هم داخلی ایران).
    *   **چندفروشندگی (Multi-Vendor):** در صورت نیاز، پیاده‌سازی قابلیت چندفروشندگی (مانند افزونه Dokan) برای اجازه دادن به کاربران برای فروش محصولات خود.
*   **پلتفرم آموزش آنلاین پیشرفته (Advanced Online Learning Platform - LearnPress Enhancement):**
    *   **کلاس‌های زنده:** امکان برگزاری کلاس‌های زنده (Live Classes) با ادغام ابزارهای ویدئو کنفرانس (مانند Zoom, Jitsi Meet یا یک راهکار بومی).
    *   **ارزیابی و آزمون:** سیستم ارزیابی و آزمون‌های پیشرفته با انواع سوالات مختلف، زمان‌بندی و نمره‌دهی خودکار.
    *   **گواهینامه‌های دیجیتال:** صدور گواهینامه‌های دیجیتال قابل اشتراک‌گذاری برای تکمیل دوره‌ها.
    *   **مسیرهای یادگیری:** قابلیت تعریف مسیرهای یادگیری (Learning Paths) برای کاربران.
*   **سیستم اهدای کمک‌های مالی (Donation System):**
    *   **جمع‌آوری کمک:** قابلیت جمع‌آوری کمک‌های مالی برای خیریه‌ها، مساجد، مراکز اسلامی، پروژه‌های فرهنگی و آموزشی.
    *   **گزارش‌دهی شفاف:** ارائه گزارش‌های شفاف از کمک‌های جمع‌آوری شده و نحوه مصرف آن‌ها.
    *   **کمپین‌ها:** امکان ایجاد کمپین‌های جمع‌آوری کمک مالی با اهداف مشخص.
*   **بخش رسانه (Media Hub):**
    *   **گالری تصاویر و ویدئو:** گالری‌های پیشرفته برای تصاویر و ویدئوهای اسلامی با قابلیت دسته‌بندی، برچسب‌گذاری و جستجو.
    *   **پخش زنده:** امکان پخش زنده (Live Streaming) رویدادهای مذهبی، سخنرانی‌ها و کلاس‌ها.
    *   **کتابخانه صوتی:** کتابخانه صوتی جامع از تلاوت قرآن، ادعیه، سخنرانی‌ها، مداحی‌ها و پادکست‌های اسلامی با قابلیت جستجو و فیلتر.
    *   **مدیریت رسانه:** ابزارهای مدیریت رسانه برای آپلود، سازماندهی و به اشتراک‌گذاری محتوا.
*   **پشتیبانی از چندزبانگی (Multilingual Support):**
    *   یکپارچه‌سازی کامل با افزونه‌های قدرتمند چندزبانگی مانند WPML یا Polylang برای پشتیبانی از زبان‌های مختلف (فارسی، عربی، انگلیسی، اردو، ترکی و غیره) به صورت بومی و کامل.
    *   اطمینان از پشتیبانی کامل RTL برای زبان‌های راست به چپ.

#### 3.3. پیشنهادات تکمیلی (Agent's Suggestions) - برای نوآوری و تمایز

*   **سیستم اعتبارسنجی و امتیازدهی پیشرفته (Advanced Gamification with GamiPress):**
    *   **امتیازدهی پویا:** توسعه سیستم امتیازدهی برای فعالیت‌های مثبت کاربران (مانند مشارکت در بحث‌ها، تکمیل دوره‌ها، کمک به دیگران، اشتراک‌گذاری محتوای مفید، پاسخ به سوالات).
    *   **نشان‌ها و رتبه‌بندی:** طراحی نشان‌های جذاب (Badges) و سیستم رتبه‌بندی (Leaderboards) برای تشویق مشارکت فعال و رقابت سالم.
    *   **جوایز مجازی/واقعی:** امکان تعریف جوایز مجازی یا حتی واقعی برای کاربران برتر.
*   **فید شخصی‌سازی شده هوشمند (Intelligent Personalized Feed):**
    *   **الگوریتم هوشمند:** پیاده‌سازی یک الگوریتم هوشمند برای نمایش محتوا و فعالیت‌های مرتبط با علاقه‌مندی‌ها، تعاملات قبلی و اطلاعات مذهبی کاربر.
    *   **فیلتر و دسته‌بندی:** قابلیت فیلتر کردن فید بر اساس موضوع، نوع محتوا، گروه‌ها و افراد دنبال شده.
*   **ادغام با APIهای اسلامی خارجی (External Islamic API Integration):**
    *   **اوقات شرعی و قبله:** ادغام با APIهای معتبر جهانی برای نمایش دقیق اوقات شرعی، جهت قبله و تاریخ‌های مهم اسلامی.
    *   **احادیث و آیات روز:** نمایش حدیث یا آیه قرآن روزانه از طریق APIهای معتبر.
    *   **مراکز اسلامی:** نمایش نزدیک‌ترین مساجد، مراکز اسلامی و فروشگاه‌های حلال بر اساس موقعیت جغرافیایی کاربر.
*   **ابزارهای مدیریت محتوای اسلامی (Islamic Content Management Tools):**
    *   **قالب‌های آماده:** ارائه قالب‌های آماده و اختصاصی برای انتشار محتوای اسلامی (مانند آیات قرآن با ترجمه، احادیث با سند، داستان‌های اسلامی، زندگینامه بزرگان) با قابلیت‌های اشتراک‌گذاری آسان در شبکه‌های اجتماعی.
    *   **بخش فتاوا و سوال و جواب:** یک سیستم پرسش و پاسخ برای مسائل فقهی و مذهبی با امکان پاسخگویی توسط کارشناسان.
*   **ماژول مدیریت حج و عمره (Hajj & Umrah Management Module):**
    *   اطلاعات جامع درباره مناسک حج و عمره.
    *   راهنمای گام به گام، نقشه‌ها و نکات سفر.
    *   امکان تشکیل گروه‌های سفر و برنامه‌ریزی مشترک.
*   **ماژول مشاوره اسلامی (Islamic Counseling Module):**
    *   ارائه خدمات مشاوره مذهبی، خانوادگی و روانشناسی اسلامی توسط کارشناسان.
    *   سیستم رزرو وقت و جلسات آنلاین.

### 4. نحوه پیاده‌سازی و ملاحظات فنی (Implementation Guidelines & Technical Considerations)

*   **توسعه تدریجی (Phased Development):** امکانات را به صورت فازبندی شده و تدریجی پیاده‌سازی کنید. ابتدا بر روی رفع باگ‌ها و پیاده‌سازی کامل UI/UX مینیمال آیفون تمرکز کنید، سپس به سراغ امکانات اولیه و در نهایت امکانات ثانویه بروید.
*   **استفاده از Child Theme:** تمامی تغییرات و کدهای سفارشی را در یک Child Theme انجام دهید تا در به‌روزرسانی‌های آینده پوسته اصلی، تغییرات شما از بین نرود و امکان نگهداری آسان‌تر فراهم شود.
*   **تست مداوم (Continuous Testing):** پس از پیاده‌سازی هر قابلیت، آن را به دقت تست کنید تا از عملکرد صحیح، عدم وجود باگ و سازگاری با سایر بخش‌های پوسته و افزونه‌ها اطمینان حاصل شود. از تست‌های واحد (Unit Tests) و تست‌های یکپارچه‌سازی (Integration Tests) استفاده کنید.
*   **مستندسازی (Documentation):** تمامی تغییرات، کدهای جدید، توابع و کلاس‌ها را به خوبی مستندسازی کنید. مستندات باید شامل توضیحات کد، پارامترها، مقادیر بازگشتی و مثال‌های استفاده باشد.
*   **سازگاری با مرورگرها (Browser Compatibility):** اطمینان از سازگاری کامل پوسته با مرورگرهای اصلی (Chrome, Firefox, Safari, Edge) و نسخه‌های مختلف آن‌ها.
*   **بهینه‌سازی برای SEO (SEO Optimization):** تمامی ساختار HTML و محتوا باید بهینه‌سازی شده برای موتورهای جستجو باشد. استفاده صحیح از تگ‌های سمانتیک HTML5، داده‌های ساختاریافته (Schema Markup) و سرعت بارگذاری بالا از اهمیت ویژه‌ای برخوردار است.
*   **قابلیت دسترسی (Accessibility - A11y):** پوسته باید مطابق با استانداردهای WCAG (Web Content Accessibility Guidelines) طراحی و کدنویسی شود تا برای افراد دارای معلولیت نیز قابل استفاده باشد.
*   **استفاده از Hooks و Filters وردپرس:** برای افزایش قابلیت توسعه و نگهداری، تا حد امکان از Hooks و Filters وردپرس برای افزودن یا تغییر قابلیت‌ها استفاده کنید.
*   **نسخه‌بندی (Version Control):** تمامی تغییرات باید از طریق Git و GitHub مدیریت شوند. هر قابلیت یا رفع باگ باید در یک شاخه جداگانه توسعه یابد و پس از بررسی (Code Review) در شاخه اصلی ادغام شود.

این پرامپت یک نقشه راه جامع و دقیق برای ارتقاء پروژه IslamNET ارائه می‌دهد. با تمرکز بر این نکات، می‌توانید یک پلتفرم قدرتمند، جذاب و مطابق با بالاترین استانداردهای فنی و طراحی برای جامعه مسلمانان ایجاد کنید.
