# IslamNET (Jamealslam / IslamRepublic)

IslamNET is a comprehensive WordPress-based social networking and educational platform designed for the global Muslim community. This project follows the architecture and pattern of the **SocialV** theme, integrating social interaction, e-learning, and media sharing.

## Features
- **Social Networking:** Built on BuddyPress for community engagement, profiles, and groups.
- **Education (LMS):** Integrated LearnPress for online courses and webinars.
- **Media Hub:** Support for audio/video streaming and digital libraries.
- **Auto-Installer:** Uses TGMPA to automatically prompt and install required plugins.
- **RTL Ready:** Full support for Persian, Arabic, and other RTL languages.

## Installation
1. Upload the `ISLAMNET` folder to your WordPress `wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.
3. Upon activation, you will be prompted to install the required plugins (BuddyPress, Elementor, LearnPress, etc.).
4. Follow the on-screen instructions to complete the setup.

## Technical Architecture
- **Base:** WordPress
- **Social Engine:** BuddyPress
- **Page Builder:** Elementor
- **LMS:** LearnPress
- **Gamification:** GamiPress
